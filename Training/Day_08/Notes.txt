Topic: Button Look & Anchor CSS Effects

====================================
BUTTON LOOK (CSS STYLING)
====================================
Buttons are styled using CSS to improve UI and user experience.

Key CSS Properties used:
- padding → controls button size
- border → button outline
- border-radius → rounded corners
- background-color → button color
- color → text color
- font-size → text size
- cursor: pointer → hand cursor on hover
- transition → smooth animation effect
- box-shadow → glow / shadow effect
- transform: scale() → zoom effect on hover

Common Button Effects:
1) Hover color change
2) Scale animation
3) Outline to filled button
4) Glow effect using box-shadow

====================================
ANCHOR (<a>) CSS EFFECTS
====================================
Anchor tags are styled to look interactive and modern using CSS.

Key CSS Properties used:
- text-decoration: none → remove default underline
- color → link color
- position: relative → for pseudo-element positioning
- ::after pseudo-element → custom underline effect
- transition → smooth hover animation
- border → box style link
- background-color → hover background

Anchor Effects Implemented:
1) Animated underline hover effect
2) Color change on hover
3) Box hover effect

====================================
CONCEPT SUMMARY
====================================
Buttons and anchor effects are used to:
- Improve UI design
- Improve user experience
- Show interactivity
- Make websites visually attractive
- Guide user actions
